GRBL Controller Plus
Professional Android application for controlling GRBL-based CNC machines. Streamline your G-code workflow directly from your mobile device.

Features
Real-time Control: Jog axes, home machine, and manage feed/spindle rates.

G-code Sender: Stream G-code files directly to your CNC controller via USB/OTG or Bluetooth.

Status Monitoring: View real-time machine coordinates and state updates.

Command Console: Send manual commands and view serial logs.

Optimized UI: Designed for quick access in workshop environments.

Getting Started
Install the app from the Google Play Store.

Connect your Android device to your CNC controller using a USB OTG cable or Bluetooth module.

Select the correct baud rate and port to establish a connection.

Compatibility
Supports GRBL v0.9, v1.1, and compatible firmware.

Tested on a wide range of CNC routers, laser engravers, and milling machines.
